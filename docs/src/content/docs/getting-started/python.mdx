Requires Python `>=3.11`.

## CLI quick start

```bash
memory init ./my-brain
memory ingest ./docs/hedgehogs.md --brain ./my-brain
memory search "where do hedgehogs live?" --brain ./my-brain
memory serve --addr 127.0.0.1:8080
```

See the [CLI reference](/reference/cli/) for the current per-SDK command matrix.

Cross-SDK eval parity today is the shared daemon surface: `ask-basic`, `ask-augmented`, and `search-retrieve-only`. Native LongMemEval replay and agentic runs are outside the Python SDK today.

## Environment variables

| Variable      | Purpose                                                                 |
| ------------- | ----------------------------------------------------------------------- |
| `JB_HOME`     | Storage root (default `~/.jeffs-brain/`).                               |
| `JB_TOKEN`    | Hosted bearer token. When set the SDK drives the platform `HttpStore`.  |
| `JB_ENDPOINT` | Hosted endpoint URL when `JB_TOKEN` is set.                             |

### Programmatic hello-world

Open an `FsStore` for persistence, attach a SQLite-backed `search.Index` for ranking, ingest a markdown snippet, then run a BM25 query. Store ops are async; index ops are sync, so the example wraps the async pieces in `asyncio.run`.

```python
# SPDX-License-Identifier: Apache-2.0
import asyncio
from pathlib import Path

from jeffs_brain_memory.knowledge import (
    CONTENT_TYPE_MARKDOWN,
    IngestRequest,
    Options,
    SearchRequest,
    new,
)
from jeffs_brain_memory.search import Chunk, Index
from jeffs_brain_memory.store.fs import FsStore

BRAIN_ID = "hello"
MARKDOWN = "# Hedgehogs\n\nHedgehogs live in hedgerows across Europe.\n"


async def amain() -> None:
    root = Path("./data") / BRAIN_ID
    root.mkdir(parents=True, exist_ok=True)

    store = FsStore(root)
    index = Index(root / "search.db")
    kb = new(Options(brain_id=BRAIN_ID, store=store, index=index))

    try:
        ingest = await kb.ingest(
            IngestRequest(
                path="hedgehogs.md",
                content=MARKDOWN.encode("utf-8"),
                content_type=CONTENT_TYPE_MARKDOWN,
            )
        )
        index.upsert_chunks(
            [Chunk(id=f"{ingest.path}:0", document_id=ingest.path, path=ingest.path, text=MARKDOWN)]
        )

        for hit in index.search_bm25("where do hedgehogs live?", top_k=3):
            print(f"[{hit.score:.3f}] {hit.path}: {hit.snippet}")
    finally:
        index.close()
        await store.close()


asyncio.run(amain())
```

See [`examples/python/`](/examples/python/) for the full walkthrough, including the in-memory BM25 fallback when no `Index` is wired.
